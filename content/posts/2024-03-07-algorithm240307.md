# 2024년 3월 7일 알고리즘 문제풀이

## 문제

- [두 정수 사이의 합](https://school.programmers.co.kr/learn/courses/30/lessons/12912)

### 난이도

Lv.1

### 코드

```python
def solution(a, b):
    high = max(a,b)
    low = min(a,b)
    answer = 0
    for i in range(low,high+1):
        answer += i
    return answer
```
