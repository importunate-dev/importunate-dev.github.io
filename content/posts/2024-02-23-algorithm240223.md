# 2024년 2월 23일 알고리즘 문제풀이

## 문제

- [짝지어 제거하기](https://school.programmers.co.kr/learn/courses/30/lessons/12973)

### 난이도

Lv.2

### 코드

```python
def solution(s):
    arr = list(s)
    stk = []
    for i in range(len(s)):
        if not stk:
            stk.append(arr[i])
            continue
        else:
            if stk[-1] == arr[i]:
                stk.pop()
            else:
                stk.append(arr[i])
    if not stk:
        answer = 1
    else:
        answer = 0
    return answer
```
