# 2023년 12월 14일 알고리즘 문제풀이

## 문제

[나머지가 1이 되는 수 찾기](https://school.programmers.co.kr/learn/courses/30/lessons/87389)

### 난이도

Lv.1

### 코드
```
def solution(n):
    answer = 0
    for i in range(2,n):
        if not (n-1)%i:
            return i
```
