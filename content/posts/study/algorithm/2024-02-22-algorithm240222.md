# 2024년 2월 22일 알고리즘 문제풀이

## 문제

- [정수 내림차순으로 배치하기](https://school.programmers.co.kr/learn/courses/30/lessons/12933)

### 난이도

Lv.1

### 코드

```python
def solution(n):
    arr = list(str(n))
    map(int,arr)
    arr.sort(reverse=True)
    map(str,arr)
    answer = ''.join(arr)
    return int(answer)
```
