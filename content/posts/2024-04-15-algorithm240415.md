# 2024년 4월 15일 알고리즘 문제풀이

## 문제
- [의상](https://school.programmers.co.kr/learn/courses/30/lessons/42578)

### 난이도

Lv. 2

### 코드

```python
def solution(clothes):
    answer = 0
    arr = dict()
    for x in clothes:
        [n,t] = x
        if t in arr:
            arr[t] += 1
        else:
            arr[t] = 1
    cnt = 1
    for a,b in arr.items():
        cnt *= (b+1)
    answer = cnt-1
    return answer
```
