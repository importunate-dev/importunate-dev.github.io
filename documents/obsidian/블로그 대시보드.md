# 블로그 대시보드

> Dataview 플러그인이 활성화된 상태에서 열면 아래 쿼리들이 라이브 표로 렌더링됩니다.

## 최근 글 15개

```dataview
TABLE WITHOUT ID
  file.link AS "글",
  date AS "날짜",
  categories AS "카테고리",
  series AS "시리즈"
FROM "content/posts"
SORT date DESC
LIMIT 15
```

## description이 비어 있는 글

```dataview
TABLE WITHOUT ID
  file.link AS "글",
  date AS "날짜",
  categories AS "카테고리"
FROM "content/posts"
WHERE !description OR description = ""
SORT date DESC
```

## 시리즈별 글 개수

```dataview
TABLE WITHOUT ID
  key AS "시리즈",
  length(rows) AS "글 수"
FROM "content/posts"
WHERE series
FLATTEN series AS s
GROUP BY s AS key
SORT length(rows) DESC
```

## 카테고리별 글 개수

```dataview
TABLE WITHOUT ID
  key AS "카테고리",
  length(rows) AS "글 수"
FROM "content/posts"
FLATTEN categories AS c
GROUP BY c AS key
SORT length(rows) DESC
```

## 태그별 글 개수 (상위 30)

```dataview
TABLE WITHOUT ID
  key AS "태그",
  length(rows) AS "글 수"
FROM "content/posts"
FLATTEN tags AS t
GROUP BY t AS key
SORT length(rows) DESC
LIMIT 30
```

## 시리즈가 없는 글

```dataview
TABLE WITHOUT ID
  file.link AS "글",
  date AS "날짜",
  categories AS "카테고리"
FROM "content/posts"
WHERE !series
SORT date DESC
```
