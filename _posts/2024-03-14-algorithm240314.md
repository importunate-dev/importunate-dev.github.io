# 2024년 3월 14일 알고리즘 문제풀이

## 문제

- [음양 더하기](https://school.programmers.co.kr/learn/courses/30/lessons/76501)

### 난이도

Lv.1

### 코드

```python
def solution(absolutes, signs):
    answer = 0
    for i in range(len(signs)):
        if signs[i]:
            answer += absolutes[i]
        else:
            answer -= absolutes[i]
    return answer
```
