# 2024년 1월 25일 알고리즘 문제풀이

## 문제

- [정수 제곱근 판별](https://school.programmers.co.kr/learn/courses/30/lessons/12934)

### 난이도

Lv.1

### 코드

```pythondef solution(n):
    tmp = int(n**(0.5))
    if tmp**2 == n:
        answer = (tmp+1)**2
    else:
        answer = -1
    return answer
```

제곱근은 0.5제곱과 같음.
